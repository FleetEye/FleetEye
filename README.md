## 🚌 Le parcours

Depuis toujours, j'ai voulu construire quelque chose. Entreprendre.
J'ai ce réflexe de chercher la solution derrière chaque problème — pas de
le contourner, de le régler.

Pendant cinq ans, conducteur de bus chez Transdev STAS à Saint-Étienne,
c'est ce réflexe qui m'a fait regarder le métier autrement : ce qui coinçait
au quotidien, ce qui pourrait mieux tourner, ce que les outils ne voyaient
pas. Le transport public vécu de l'intérieur, côté volant.

## 💡 Le déclic

Les SAEIV actuels traitent le conducteur comme un périphérique du système.
Après cinq ans à ce poste, je sais que le poste de conduite est la première
interface d'un réseau — pas la dernière. FleetEye est construit dans ce sens.

## 🛠️ Les standards

J'ai découvert NeTEx à la source : le XML brut, les schémas officiels, le
profil France. Le standard est vaste, je n'en ai pas fait le tour — mais
j'en ai compris la logique, et le reste s'apprend en construisant.

Ici, ce n'est pas une contrainte réglementaire traitée en bout de chaîne :
c'est le socle.
